# 9.1 餘額通

## 9.1A 問：設置了最小贖回份額為整數，為什麼能贖回小數份額？

答：餘額通標的，贖回時只判斷持股份額，不判斷最小賣出份額/金額限制。
自動贖回是用於償還欠款或交收的資金，如果無法自動贖回會導致客戶欠款。所以餘額通贖回時不受最小賣出份額/金額的限制。
