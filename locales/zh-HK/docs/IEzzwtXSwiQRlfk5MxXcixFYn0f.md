# 運營系统（OPA）

OPERATION-ADMIN（簡稱 OPA）運營系統是 Whale 針對劵商客戶營銷所支持的 SaaS 營銷運營中台，它是在證劵系統的業務數據經過 Whale 數據中台的數據整理與標籤優化後，提供完整的證劵客戶營銷服務管理，包含各類卡券、權益、獎勵、投放、活動、頁⾯搭建等等營銷常⽤功能。全量的營銷營中台囊括了劵商客戶行銷全量⽣命週期中的各個節點，對目標群體輸出完整特定的營銷服務。
