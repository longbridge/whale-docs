# 賬戶明細

## 適用場景

查詢帳戶發生的資產流水明細，並支持分幣種、業務碼等多維度篩選查詢。

## 前置条件

無

## 操作說明

菜單入口：資產賬戶&gt;帳戶&gt; 帳戶明細

選擇需要查詢的資產類型，分現金、股票、基金、期權、債券、OTC 其他、虛擬資產
