# 快速上手

# 概述

WTT 客戶列表、客戶信息組件可讓交易員一站式存取客戶資產、歷史交易、代理關係和帳戶流水等詳細資訊。

# 快速上手

通過 WTT 右上角組件工具箱 - 客戶分類中找到這兩個組件，可將組件拖動至面板或獨立使用。
