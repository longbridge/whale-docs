# 2024-12-12 更新日誌

# 🎉 新功能

- 新增 HK 席位號的管理功能

    - 若所有賬戶用相同的席位號，則無需單獨配置
    - 路徑 1：客戶管理系統 - 賬戶管理 - 工單中心 - 客戶/賬戶信息 - 修改席位號
    - 路徑 2：客戶管理系統 - 賬戶管理 - 賬戶列表
    - 權限：account.update_broker_id
