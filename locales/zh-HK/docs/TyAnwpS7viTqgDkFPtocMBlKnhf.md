# 【場景】如何創建一個 APP Push 消息？

## 適用場景

介绍本文檔的應用場景，可以給哪類群體提供什麼操作，解決什麼訴求

## 前置條件

無

## 使用说明

投放系統的觸發配置提供了 Push 的自定義模板功能，可以在配置計劃時直接配置內容

以下是創建 App Push  的步驟順序：

- 選擇通道爲 Push
- 設置標題、內容、上傳封面圖片
- 選擇通知行爲：常用的爲打開某個活動鏈接或跳轉到新股認購的頁面，具體可按需選擇
