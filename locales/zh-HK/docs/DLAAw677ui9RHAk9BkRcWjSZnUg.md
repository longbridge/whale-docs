# 業務介紹

# 概述

公司行動系統是證券後臺的重要組成部分，主要用於處理公司股權變動相關業務，包括分紅、拆合股、換股等。

目前，公司行動系統已支持牛熊證回收、拆合股、分紅、供股、邀約、ADR 等多類型。WTT 前台公司行動頁面僅支持查看客戶相關處理明細，咱不支持操作處理相關業務。
