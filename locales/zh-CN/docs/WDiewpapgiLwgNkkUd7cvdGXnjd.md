#  会计中台

## 适用场景

用户在进行财务记帐时，需依照客户实际发生的资金活动进行记帐、对帐、分析等工作。系统提供了一套可以自动获取财务流水数据源，并且按照预设的科目以及分录规则自动进行分录记帐：

1. 获取各种数据源，比如客户变动明细、银行帐单、用户收费帐单、管道帐单； 
2. 支持自定义会计科目属性； 
3. 支持自定义会计科目； 
4. 支持自定义分录规则； 
5. 自动形成科目明细帐； 
6. 自动形成科目余额汇总表；
