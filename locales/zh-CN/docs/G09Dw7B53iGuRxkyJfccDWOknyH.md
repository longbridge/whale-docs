# 9.1 余额通

## 9.1A 问：设置了最小赎回份额为整数，为什么能赎回小数份额？

答：余额通标的，赎回时只判断持股份额，不判断最小卖出份额/金额限制。
自动赎回是用于偿还欠款或交收的资金，如果无法自动赎回会导致客户欠款。所以余额通赎回时不受最小卖出份额/金额的限制。
