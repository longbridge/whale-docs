# 系统介绍

活动系统是 Whale OPA 运营系统所提供营销活动管理核心，方便劵商租户来管理各种市场营销活动，提供各种营销活动的配置管理，系统目前提供下列活动类型组：

透过方便的营销活动模板，来管理各种营销活动。

<img src="/assets/X73CbSkaIo7tLSxomLrcalD3ngc.png" src-width="1276" src-height="920" align="center"/>
