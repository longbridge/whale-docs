# 常见问题

<div class="callout callout-bg-2 callout-border-2">
<div class='callout-emoji'>❓</div>
<p>问：为什么在新建的时候选不到我想要新增的银行卡？</p>
</div>

答：先透过款项管理-APP 管理 - 卡所属银行&用户绑卡 - 国家/地区 tab 确定卡的属性资讯是否正确维护，如果维护正确仍然没有，可能是因为政策法规限制，不支援该国家的银行卡。
