# 快速上手

# 概述

WTT 客户列表、客户信息组件可让交易员一站式存取客户资产、历史交易、代理关系和帐户流水等详细资讯。

# 快速上手

通过 WTT 右上角组件工具箱 - 客户分类中找到这两个组件，可将组件拖动至面板或独立使用。
