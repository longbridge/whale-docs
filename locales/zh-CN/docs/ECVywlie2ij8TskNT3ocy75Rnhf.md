# 消息中心

消息中心是集中处理和管理企业和使用者之间各种讯息的系统。消息中心为业务和使用者之间建立起了讯息传送的桥梁，使得各类讯息可以在合适的场景触达到使用者端。讯息范本可以很好地管理不同业务需求场景下的各种范本，规范讯息发送的格式；讯息通道对接了简讯、邮件等各种通道，提供开放的服务给各个业务方进行调用触发，由消息中心进行审核，将审核通过的讯息发送到使用者端。
