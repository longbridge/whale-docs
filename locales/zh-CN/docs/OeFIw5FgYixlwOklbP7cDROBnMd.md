# 2025-08-12 更新日志

# 🪀 改进与修复

- 支持自订金额为客户完成换汇：
    - 针对客户换汇申请，操作人员可在后台指定金额进行处理。
    - 路径：WBO - 款项管理 - 换汇 - 客户换汇
    <img src="/assets/L7cAb7TD6o3SR9xdqyTcOngzn3e.png" src-width="3294" src-height="1750" align="center"/>

- 支持手工调账业务码管理：
    - 如手工调账需增减业务码，可在业务码分组进行管理。
    - 路径：WBO - 资产账户 - 业务码 - 业务管理
