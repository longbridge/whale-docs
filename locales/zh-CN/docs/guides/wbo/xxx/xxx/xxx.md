# 账户明细

## 适用场景

查询帐户发生的资产流水明细，并支持分币种、业务码等多维度筛选查询。

## 前置条件

无

## 操作说明

菜单入口：资产账户&gt;账户&gt; 账户明细

选择需要查询的资产类型，分现金、股票、基金、期权、债券、OTC 其他、虚拟资产
