# 非标准活动管理

## 适用场景

想要直接给用户以某个活动名称进行发奖励时，可以先创建一个活动名称

使用人员：运营

使用场景：线下活动发奖，例如：大客户回馈之类的无具体活动规则，只是为了单纯给用户发放奖励的活动

主要功能：创建/管理活动程式码

## 前置条件

无

## 操作说明

菜单入口：活动系统>非标准活动管理

- 点击页面上方新建按钮，打开如下页面（页面重构中，后面补充）
