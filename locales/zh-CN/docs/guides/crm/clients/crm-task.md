# 任务

## 适用场景

本作业将客户管理从完成从新用户转成一个有效户的目标推进过程中，以集成汇总与附予任务方式来管理每个与用户（客户）的互动进程，每个进程都可以方便追踪管理，同时在跟进（记录互动）时也是以卡片方式来展是用户（客户）的全局记录

## 前置条件

无

## 操作说明

菜单入口：客户管理系统&gt;CRM&gt;任务

每个不同的互动进程的信息展示。例如，入金/出金/注册等。
