# 业务介绍

# <b>概述</b>

会计中台提供了一套可以自动获取数据源，并且按照预设的科目以及分录规则自动进行记账，进而完成账实核对，整套系统支持的功能如下：

1. 自动获取各种数据源形成明细分录和合并分录，数据源包括支付流水、客户收费账单、银行账单等；
2. 支持后台自定义会计科目属性；
3. 支持后台自定义会计科目；
4. 支持后台自定义分录规则；
5. 支持后台自定义资金类型；
6. 支持后台自定义合并规则；
7. 支持后台自定义是否记账的业务类型；
8. 形成科目明细账；
9. 形成科目余额汇总表；
