# 2024-12-12 更新日志

# 🎉 新功能

- 新增 HK 席位号的管理功能
    - 针对不同证券账号，支持分别配置不同的 HK 席位号
    - 若所有账户用相同的席位号，则无需单独配置
    - 路径 1：客户管理系统 - 账户管理 - 工单中心 - 客户/账户信息 - 修改席位号
    - 路径 2：客户管理系统 - 账户管理 - 账户列表
    - 权限：account.update_broker_id
