# 系統介紹


# 一、系統介紹


本指南針對加密貨幣出幣業務的後台操作全流程，涵蓋記錄生成、查詢跟蹤、工單審批流程，確保後台人員規範操作，保障業務合规與資金安全。


# 前置条件

- 後台操作人員使用專屬賬號密碼登錄 WBO 後台系統，進入「虛擬資產管理 - 出入幣 - 出幣」模塊。
- 確認賬號已具備「出幣申請」「審批」等操作權限，無權限需聯繫管理員申請。

# 二、操作說明

在客戶出幣前，需要進行出幣加密貨幣設置。

## 客户錢包地址

![image.png](/assets/4c7d5af9d6592a4fa5d04e13283bb8f8.png)


## 出幣**操作**


**請求接收與記錄生成**


> 📍 菜單入口：WBO - 虛擬資產管理 - 出入幣 - 出幣申請

- 客戶端提交出幣請求後，系統自動接收請求信息，並生成唯一入幣記錄（含記錄編號、用戶證券賬號、入幣幣種、金額、目標地址、交易哈希 TXID、請求時間、狀態等核心信息）。

![image.png](/assets/38e0a60bd0ff312a73bc112a47b28c9e.png)

- 客戶提交的出幣申請已自動對接 WBO 工單（工單標識：atm.va.coin_withdraw_apply），系統將依據預設的審批配置流開展審批。

![image.png](/assets/4e0974a527da53c6bc27217662768aba.png)
