# 快速上手


# 概述


入币是将加密货币（如比特币、以太坊等）从外部地址（其他交易平台、他人钱包等）转入自身数字钱包或交易平台账户的过程，以下是标准化操作步骤，确保操作准确与资产安全：


# 快速上手


入币前完成钱包地址认证是保障资产安全的关键前置环节。


# 前置条件


您需要获取以下授权后方可正常使用系统功能


| 权限名称                   | 权限说明                |
| ---------------------- | ------------------- |
| 管理「加密货币」、「入币申请」、「入币记录」 | 管理加密货币、入币申请、入币记录的权限 |


# 加密货币**配置**


在客户入币前，需要进行入币加密货币设置。

# 客户钱包地址


入币前完成钱包地址认证是保障资产安全的关键前置环节。


![image.png](/assets/e37b33fc6b34d71ad6d5e26ee7ab31e4.png)


# **认证后入币操作简述**


> 📍 菜单入口：WBO - 虚拟资产管理 - 出入币 - 入币申请

- 客户端提交入币请求后，后台系统会自动生成对应的入币记录，便于后台操作人员查询跟踪全流程；若遇异常或有问题的入币申请，后台操作人员可直接在系统中执行删除操作

![image.png](/assets/dfaa0e237a847e9970ab77270eec47dc.png)


> 📍 菜单入口：WBO - 虚拟资产管理 - 出入币 - 入币记录

- 可通过目标钱包或平台的“入币记录”模块，查看交易状态、区块确认次数等详情，跟踪资产到账情况。

![image.png](/assets/6644b9e53fd080a7cfd3193793b9b39e.png)
