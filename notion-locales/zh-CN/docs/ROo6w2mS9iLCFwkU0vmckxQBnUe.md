# 2025-02-24 更新日志


# 🎉 新功能

- 新增客户多头股票保证金变更记录
    - 1、股票停复牌自动处理客户保证金
    - 2、新增/编辑/系统停复牌/到期失效均会记录客户保证金变更记录
    - 路径：风控管理 - 保证金 - 客户保证金 - 多头股票
    - 权限：客户保证金变更记录查询 `credit_management.client_margin_change_query`
