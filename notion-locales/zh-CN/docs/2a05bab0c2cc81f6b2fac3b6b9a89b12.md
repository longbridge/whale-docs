# 业务介绍


# 概述


客户在使用「存入加密货币」「提取加密货币」功能时，所使用的钱包地址均需完成认证及加白流程，具体规则如下：

1. 认证方式：目前仅支援「小额支付验证」—— 客户需透过向指定地址完成小额加密货币支付，验证钱包地址的归属权，完成认证后方可进入加白流程。
2. 加白流程：客户需先提交钱包地址认证申请，经系统校验通过后，该地址将被纳入「加白名单」，成为可用于加密货币出入的合法地址。
3. 后台管理：所有已完成加白的客户钱包地址，均会在后台系统统一维护（包含地址资讯、认证时间、状态等），支援即时查询与管理，确保出入币操作的合规性与安全性。透过上述流程，可有效验证钱包地址的真实性与归属权，保障客户加密货币出入环节的资金安全。
