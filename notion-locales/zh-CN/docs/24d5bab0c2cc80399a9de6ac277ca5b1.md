# 2025-08-12 更新日志


# 🪀 改进与修复

- 支持自订金额为客户完成换汇：
    - 针对客户换汇申请，操作人员可在后台指定金额进行处理。
    - 路径：WBO - 款项管理 - 换汇 - 客户换汇

    ![image.png](/assets/dfc54e09444b420f2bfd56d43f07b1ea.png)

- 支持手工调账业务码管理：
    - 如手工调账需增减业务码，可在业务码分组进行管理。
    - 路径：WBO - 资产账户 - 业务码 - 业务管理

    ![image.png](/assets/a3efa14a7d86ab1fdc857be311f91ae0.png)
