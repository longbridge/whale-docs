# 业务介绍


# 概述


出币（又称「提币」「放币」）是指用户将存放在交易平台或个人钱包中的加密货币，转出至外部目标地址（如其他交易平台账户、个人数字钱包、他人收款地址等）的资产转移行为。其核心特点与场景如下：

- **业务本质**：通过区块链网络广播交易，实现数字资产所有权的跨地址转移，需依赖平台审核与链上确认双重机制保障安全。
- **常见场景**：资产划转（平台间转移）、个人储存（转入硬件钱包）、交易支付（转至对方地址）、法币兑现（先转至支持提现的平台）等。
- **关键规则**：需匹配币种对应的区块链网络（如 USDT 的 ERC-20/TRC-20），且受平台 KYC 认证、提币限额、安全验证等规则约束。

# 出币步骤


出币业务的基本流程：

1. 前置准备：确认基础条件
2. 发起出币申请（以交易平台为例）
3. 完成安全验证与提交
4. WBO 工单审批流程
5. 等待平台审核与链上确认
6. 确认到账状态

> 📍 **注意：** 虚拟币帐户以「加密货币 + 计价货币」组合标的记帐：入币、出币时按加密货币单独记录，最终同步至证券帐户时，以该组合标的入帐（例：加密货币 ETH，预设计价货币 USD，入帐标的为「VA/HAS/ETHUSD」）。出帐时优先抵扣可用余额较大的币种，若余额不足则自动从次优先级币种扣减。
