# 快速上手


# 概述


所有已完成加白的客户钱包地址，均会在后台系统统一维护（包含地址资讯、认证时间、状态等），支援即时查询与管理，确保出入币操作的合规性与安全性。


# 快速上手


> 📍 菜单入口：WBO - 业务参数设置 - 资金参数 - 加密货币

- 前置准备：加密货币资料维护（所有操作首要步骤）

![image.png](/assets/3e5954e65ac2a2e758dc287686325d55.png)


> 📍 菜单入口：WBO - 虚拟资产管理 - 出入币 - 渠道存管账户

- 客户发起钱包地址认证前，系统会自动依据人工导入的「渠道存管账户」，为客户分配对应的账户。分配结果将做为后续认证、出入币操作的基础关联资讯，确保资金流向可追溯。

![image.png](/assets/2f08422d313a89aa22fac3c55fc7378d.png)


> 📍 菜单入口：WBO - 虚拟资产管理 - 出入币 - 客户钱包地址

- 客户提交认证通过的钱包地址后，系统会将其纳入「加白名单」。所有已加白的地址将在后台统一维护，包含地址资讯、分配的存管账户、认证时间、状态等，支援即时查询与管理，保障操作合规与资金安全。

![image.png](/assets/d894180b8875110ee001a007f105ca47.png)
