# 业务介绍


# 概述


入币是指将加密货币（如比特币、以太坊等）从一个地址（例如其他交易平台、他人钱包）转移至自身数字钱包或交易平台账户的过程，其核心依托区块链技术实现资产安全转移，且需完成钱包地址认证保障操作合规与资金安全。


# 入币步骤

1. 打开目标平台 / 钱包，进入「充值 / 入币」页面；
2. 选择对应币种与网络（务必与转出方一致）；
3. 复制生成的入币地址（核对无误）；
4. 登录转出方，粘贴地址、填写金额，完成安全验证后提交；
5. 等待区块确认，在目标平台 / 钱包查看到账状态。
