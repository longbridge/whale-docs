# SFTP 软件操作指引


# 0. 准备

- FTP 工具 Filezilla 下载地址：https://www.filezilla.cn/download (选取客户端)
- 长桥提供：
    - SFTP 主持人：`sftp.longbridgewhale.com`
    - 企业编号
    - 密钥文件

# 1. 设置站点


![image.png](/assets/7e430e63010d84ada0a5dab03d8ae558.png)


选择 SFTP 协议


主机输入长桥提供的

- 主持人：`sftp.longbridgewhale.com`
- 端口： `22`

登陆类型选择长桥提供的密钥文件
