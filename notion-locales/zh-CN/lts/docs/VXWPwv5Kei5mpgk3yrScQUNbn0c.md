# 2024-02-04 更新日志


# 🎉 新功能

- KYC 开户资料新增一键导出文件的功能
    - 路径：「客户管理系统」 - 「KYC」-「开户列表」
    - 在开户资料的详情页面，点击‘文件下载’按钮，系统将自动把开户资料中的文件信息压缩到一个文件夹中，可自行导出
    - 文件信息包括：证件照片、地址证明、转账证明、开户文件等
