# System Overview

# System Introduction

# Overview

The Announcement Management function primarily facilitates brokerage operations personnel in issuing system-related announcements to customers via trading terminals such as the APP.

# Announcement Management Description

This task is mainly used to maintain system announcements for customer-facing terminals.

<img src="/assets/BufbbH05EoUvRzx8y1YjJKTZp4f.png" src-width="3212" src-height="1612" align="center"/>

When you need to create an external announcement, click the [New Announcement] button in the upper right to begin configuring a new announcement.

<img src="/assets/C0QSbmOEhoueZYxCldLjUeBdpdd.png" src-width="2134" src-height="1518" align="center"/>

Announcement types supported are as follows:

<img src="/assets/WsAkbjChuotrGixGTwCjOyWupJc.png" src-width="943" src-height="492" align="center"/>

Supported markets:

- For announcement types: General Announcement, Regular Market Closure, Temporary Market Closure, and Channel Issues are supported.

<img src="/assets/EnZpbz1QQo5s8yx07xzj3ZnzpEb.png" src-width="1118" src-height="676" align="center"/>

Target scope

- For the announcement type "Single Stock," configuration is supported.

<img src="/assets/ONPYbzOvJoNB2Rx07IdjaD1iprf.png" src-width="1122" src-height="684" align="center"/>

Page scope:

<img src="/assets/H7eQbkuHnobNKGxnI5cjtz6ipbc.png" src-width="933" src-height="553" align="center"/>

Other announcement content can be configured as required:

<img src="/assets/LzsybobyAok1yHxzDBpjtx5rpFe.png" src-width="963" src-height="1380" align="center"/>

After adding a new system announcement:

1. Newly created announcements must be submitted for work order approval and will take effect only after approval.
2. You can perform [Edit] or [Disable] actions in the action area on the right side of the record.
