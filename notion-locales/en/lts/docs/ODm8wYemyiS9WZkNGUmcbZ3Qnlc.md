# Quick Start — Bull and Bear Certificates, Bond Redemption, Privatization (TM)

# Quick Start - Bull/Bear Certificates, Bond Recalls, Privatization (TM)

# Applicable Scenarios

- Tenants directly connected to HKEX (with 05 file)
- TM includes Bull/Bear Certificate recall (including residual value), privatization, and prerequisites for bond recalls
- Equity recovery and residual value return are executed simultaneously

# Operational Instructions

## 1. Creation of Notices and Plans

- Settlement staff import the previous afternoon’s 05 and 02 files daily
    - Brokers executing Corporate Actions that same evening should import the files that evening
    - Brokers executing Corporate Actions the next day should import the files the following morning
    - After successful import and parsing, the system will run an automated creation process, which can complete in approximately 10 minutes

- The system will, based on the imported 05 file, generate Corporate Actions with positions on the record date

## 2. Registration

- Operations on Corporate Actions may begin on the evening of, or the day after, the CCASS residual value payout (CCASS payment date)
    - Example: If the CCASS payment date = 2024-02-05, execution may occur on the evening of the 5th or the morning of the 6th

- Click [Register]. Path: Corporate Action &gt; Corporate Action &gt; Details

## 4. Execution

- After successful registration (or completion of tail adjustments), click Submit Execution to begin executing the Corporate Action
- Execution consists of two steps — Submit Execution and Execution Review — dual operation is recommended

Click Submit Execution
