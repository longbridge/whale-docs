# 通道管理

### 通道管理

通道管理是針對與  APP 常見交互方式， 例：Banner 橫條、App 彈窗、App Push、郵件、短信等通道內容管理。

其中橫幅廣告（Banner），是個呈現一個廣告推廣內容的圖片，放置在 App 等互聯網頁面上，是互聯網廣告中最基本的廣告形式，一般是使用 GIF 格式的圖像檔，可以使用靜態圖形，或新興的 Rich Media Banner（豐富媒體 Banner）能賦予橫幅更強的表現力和交互內容。
