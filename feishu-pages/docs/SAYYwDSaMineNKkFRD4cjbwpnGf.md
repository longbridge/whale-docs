# 業務介紹

# 概述

權益系統是 Whale 來管理各類營銷獎勵與權益的發放，包含各類卡劵的創建與發放管理，整個 Whale 權益系統體系中，權益主要包括卡券福利、獎勵中心、權益中心三大作業功能。
