# 業務碼查詢

## 適用場景

該菜單支持查詢所有的業務碼，同時可以更改對客展示的業務碼名稱以及業務碼對應的流水是否對客戶展示。

每一筆流水都會對應一個業務碼，業務碼是業務場景的唯一標識。

## 前置條件

無

## 操作說明

菜單入口：資產賬戶&gt;業務碼&gt;業務碼查詢

進入菜單頁，可以在列表查詢所有業務碼，支持通過編碼和名稱搜索
