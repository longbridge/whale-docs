# 業務介紹


# 概述


客戶在使用「存入加密加密貨幣」「提取加密貨幣」功能時，所使用的錢包地址均需完成認證及加白流程，具體規則如下：

1. 認證方式：目前僅支援「小額支付驗證」—— 客戶需透過向指定地址完成小額加密貨幣支付，驗證錢包地址的歸屬權，完成認證後方可進入加白流程。
2. 加白流程：客戶需先提交錢包地址認證申請，經系統校驗通過後，該地址將被納入「加白名單」，成為可用於加密貨幣出入的合法地址。
3. 後台管理：所有已完成加白的客戶錢包地址，均會在後台系統統一維護（包含地址資訊、認證時間、狀態等），支援即時查詢與管理，確保出入幣操作的合規性與安全性。透過上述流程，可有效驗證錢包地址的真實性與歸屬權，保障客戶加密貨幣出入環節的資金安全。
