# 快速上手


# 概述


入幣是將加密貨幣（如比特幣、以太坊等）從外部地址（其他交易平台、他人錢包等）轉入自身數字錢包或交易平台賬戶的過程，以下是標準化操作步驟，確保操作準確與資產安全：


# 快速上手


入幣前完成錢包地址認證是保障資產安全的關鍵前置環節。


# 前置條件


您需要獲取以下授權後方可正常使用系統功能


| 權限名稱                   | 權限說明                |
| ---------------------- | ------------------- |
| 管理「加密貨幣」、「入幣申請」、「入幣記錄」 | 管理加密貨幣、入幣申請、入幣記錄的權限 |


# 加密貨幣**配置**


在客戶入幣前，需要進行入幣加密貨幣設置。


![image.png](/assets/05d745c5277a2a4b570f273ecc33e8e6.png)


# 客户錢包地址


入幣前完成錢包地址認證是保障資產安全的關鍵前置環節。


![image.png](/assets/e37b33fc6b34d71ad6d5e26ee7ab31e4.png)


# **認證後入幣操作簡述**


> 📍 菜單入口：WBO - 虛擬資產管理 - 出入幣 - 入幣申請

- 客戶端提交入幣請求後，後台系統會自動生成對應的入幣記錄，便於後台操作人員查詢跟蹤全流程；若遇異常或有問題的入幣申請，後台操作人員可直接在系統中執行刪除操作

![image.png](/assets/dfaa0e237a847e9970ab77270eec47dc.png)


> 📍 菜單入口：WBO - 虛擬資產管理 - 出入幣 - 入幣記錄

- 可通過目標錢包或平台的 “入幣記錄” 模塊，查看交易狀態、區塊確認次數等詳情，跟蹤資產到賬情況。

![image.png](/assets/6644b9e53fd080a7cfd3193793b9b39e.png)
