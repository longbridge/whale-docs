# 業務介紹

# 概述

入幣是指將加密貨幣（如比特幣、以太坊等）從一個地址（例如其他交易平台、他人錢包）轉移至自身數字錢包或交易平台賬戶的過程，其核心依托區塊鏈技術實現資產安全轉移，且需完成錢包地址認證保障操作合规與資金安全。

# 入幣步骤

1. 打開目標平台 / 錢包，進入「充值 / 入幣」頁面；
2. 選擇對應幣種與網絡（务必與轉出方一致）；
3. 複製生成的入幣地址（核對無誤）；
4. 登錄轉出方，粘貼地址、填寫金額，完成安全驗證後提交；
5. 等待區塊確認，在目標平台 / 錢包查看到賬狀態。
