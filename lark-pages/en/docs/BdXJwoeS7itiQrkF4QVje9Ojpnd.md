# Business Overview

# Overview

Trading order-related functional components enable users to perform standard order placement, manual orders, odd-lot matching, bulk order cancellations, basket trading, algorithmic trading, and institutional order trading through the trading components. The components also support viewing trading-related documentation and statistical data.

# Key Features

- Order placement and order inquiry (current-day orders, historical orders)
- Manual orders: submit manual orders and view manual order transaction records; supports operations to accept manual order messages from other brokers
- Odd-lot matching in the Hong Kong stock market
- Basket trading: place bulk orders by configuring basket batches
- Algorithmic trading: submit algorithmic orders via WTT, with order slicing using TWAP and VWAP algorithms
- Grid orders: view grid orders placed by client users to facilitate monitoring of order status and assessment of order placement
- Institutional orders: support direct order placement via WTT or FIX integration, enabling the splitting and allocation of institutional parent orders
