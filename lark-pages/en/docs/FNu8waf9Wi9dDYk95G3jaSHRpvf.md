# Frequently Asked Questions

<p>â“ Q: If I forget to set the broker notification status to &quot;Notified&quot; when submitting an application, what should I do?</p>
</div>

A: On the withdrawal request page, click [Edit] in the Actions column, and check "Notified" in the modal dialog.

Path: Securities Deposit &gt; Deposit Restrictions &gt; [Edit]

<div class="callout callout-bg-5 callout-border-5 callout-color-1">
<div class='callout-emoji'>ğŸ</div>
<p>â“ Q: What is the difference between the [Reject] function on the withdrawal request page and the [Delete] function on the withdrawal details page?</p>
</div>

A: They apply to different scenarios. [Reject] primarily addresses whether the submitted information as a whole is correct, for example, if the account was entered incorrectly. [Delete] is used when the application-level information is correct but a specific detail line does not need to be processed for transfer.
