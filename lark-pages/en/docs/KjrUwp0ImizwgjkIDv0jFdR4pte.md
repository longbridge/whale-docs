# Quick Start

# Overview

The Whale system supports fund transactions across three terminals—the LongPort app, WTT, and the WBO backend. This module explains how to record a fund transaction order (including private and public funds) as a subscription or redemption via the WBO backend.

# Quick Start

## Fund Order Placement

### Step 1: Navigate to the Trading Panel — Fund Order

<img src="/assets/YUMIbsP4aoF15axni6uj2eSKpbd.png" src-width="3282" src-height="1712" align="center"/>

### Step 2: Select a Customer

<img src="/assets/TS0dbVL5FoYEKlxTDYcjM7GSp0c.png" src-width="3322" src-height="1708" align="center"/>

- You can search for the target customer using the securities account number, customer ID, mobile phone number, login email, name, or other criteria.
- Upon selecting a customer, their basic information, including mobile phone number and available buying power, will be displayed.

### Step 3: Fill in Subscription/Redemption Information

<img src="/assets/UpXvbwYjToLlQYxiLUPjJ2rPpte.png" src-width="3258" src-height="1714" align="center"/>

- Complete the subscription or redemption details, including product, channel, transaction direction, amount, and subscription fee(s), then submit to finalize the order.
