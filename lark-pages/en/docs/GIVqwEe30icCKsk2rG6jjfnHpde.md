# 20. Issues with the Operations System

# 20. Operational System Issues

# 20A Q: After a customer changes their login phone number, can a market data card previously purchased continue to be used?

A: A user's market data cards are bound to the User ID; changing the login phone number will not affect the user's ability to use existing market data cards.

# 20B Q: Why does uploading the "Batch Coupon Distribution" template file fail?

A: The "Activity Code" and "Prize Code" fields in the template file may contain only English letters and numbers; entering Chinese characters or special symbols will cause the import to fail.

# 20C Q: In the campaign system, how does the "User Region" in user segmentation filter customers?

A: Filtering is based on the following fields:

Priority 1: Residence Country/Region

Priority 2: Phone Country Code

Priority 3: IP location at registration

# 20D Q: How to send an App Push to specified customers

A:

1. First, create the specified customer segment in "User Segments".

<img src="/assets/XCeWbsXvOoNKgqxA8LqjGzf9p9e.png" src-width="2826" src-height="870" align="center"/>

1. Then, create a plan under "Operation Plans".

<img src="/assets/UwFhbHychoV1XGxMG8HjxG2mpvd.png" src-width="2840" src-height="880" align="center"/>

1. Select "Scheduled - One-time" and specify the "Trigger Time".

<img src="/assets/BNm5b1EFIoWo6hxgBbAjOwR8pHc.png" src-width="2868" src-height="1042" align="center"/>

1. Select the user segment.

<img src="/assets/N330bAv3Xougt4xweWrjUtPJpxd.png" src-width="2852" src-height="1436" align="center"/>

1. Choose "Push" and enter the content.

<img src="/assets/QidDbR5X8oUhRYxKe1djKHEIpUh.png" src-width="2842" src-height="1398" align="center"/>

1. After saving and approval, click "Enable" in the "Plan List".

<img src="/assets/Exlqb5Pcfo6yJwxkevvj5u3xpBg.png" src-width="2790" src-height="1454" align="center"/>
