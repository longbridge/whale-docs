# Account Management

The Account Management module, from the perspective of accounts, assists back-office personnel in more effectively managing and serving brokerage clients, including:

1. Account List: View account status and account information, and manage account-related permissions.
2. BCAN Code Management: Query and manage clients' BCAN data and perform submissions and related operations.
3. Work Order Center: Execute data modifications, password resets, status changes, and other operations under a customer ID or account ID.
4. Record Query: Query operation records for managed accounts and device registration records.
