# Quick Start

# Overview

The Whale Accounting Middle Platform fully supports brokerage securities business accounting processes, including end-to-end management of accounting parameters, accounting entries, account balances, account detail ledgers, and accounting data sources.

For brokerage securities business, once the accounting parameters have been configured, the system supports automatic retrieval of data sources and the generation of accounting entries for exportable reports.

# Quick Start

## Prerequisites

You must obtain the following authorizations to use the system functions properly.
