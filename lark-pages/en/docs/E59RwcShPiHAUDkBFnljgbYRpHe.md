# Accounting Middle Platform

# Applicable Scenarios

When performing financial bookkeeping, users must record, reconcile, and analyze transactions in accordance with the client's actual cash activities. The system provides a solution that can automatically obtain financial transaction data sources and automatically post journal entries according to preset accounts and posting rules:

1. Automatically obtain various data sources to generate detailed and aggregated journal entries; data sources include payment transactions, customer billing invoices, bank statements, and others.
2. Supports backend customization of accounting account attributes.
3. Supports backend customization of accounting accounts.
4. Supports backend customization of posting rules.
5. Supports backend customization of fund types.
6. Supports backend customization of aggregation rules.
7. Supports backend customization of business types with respect to whether they are posted to the ledger.
8. Generates detailed account ledgers.
9. Generates account balance summary reports.
