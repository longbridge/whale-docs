# What's New — Q4 2025

# 2025-Q4 What's New

# v 2.36.1

# 🎉 New Features

- Added Balance Connect–related holdings information; customers' Balance Connect holdings can be viewed in the Customer Information component.

    <img src="/assets/JwP7bRnK9oDD2lx59qRj2LKGpSh.png" src-width="3212" src-height="424" align="center"/>

- When creating a new layout, you can choose either a Canvas layout or an Adaptive layout with one click
When opening a new tab to create a blank layout, you can select either a Canvas layout or an Adaptive layout with a single click according to your needs, making the process more convenient and intuitive.

    <img src="/assets/ZVZ9bbQPkodoEUxMbNGjp9nSpOg.png" src-width="3372" src-height="1662" align="center"/>

- Added fund and over-the-counter derivatives holdings information in the Holdings component.

    <img src="/assets/UdUobuQ36oGTDHxHMVfj4jxvpqg.png" src-width="2874" src-height="876" align="center"/>

- Per-trade details now include a time-period toggle, allowing users to view executed trades for Night Session, Pre-market, Regular Session, After-market, or All periods; an "Exchange" column has also been added to help users quickly identify the trading market for each execution, enabling more precise interpretation of market dynamics.

# 📌 Improvements and Fixes

- Improved the Order Placement component's handling of trading restrictions for suspended stocks.
- Various technical and interaction optimizations
