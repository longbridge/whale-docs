    - Path 1: Payment Management -&gt; Deposits -&gt; Deposit Request
    - When matching deposit requests for cheque deposits, you can choose to lock the funds based on business requirements. If locked, the estimated release time can be manually adjusted.
    - Path 2: Payment Management -&gt; Deposits -&gt; Deposit Match
    - When a cheque deposit request is under review, the system will indicate whether the funds are locked and display the relevant details..
    - Path 3: Payment Management -&gt; Deposits -&gt; Review Deposit
    - Locked cheque deposits can be manually released in WBO before the "Estimated Release Time" or automatically released by the system.
    - Path 4: Payment Management -&gt; Deposits -&gt; Deposit Records
    - If a manual release request is submitted, it must go through a ticket approval process.
