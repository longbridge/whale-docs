# 快速上手

可通過側邊欄快速進入「公司行動」-「客戶明細」頁面，該頁面僅支持查看參加公司行動的客戶信息，支持查看行權相關信息及詳情，為交易員提供客戶行權參考。如需操作客戶公司行動分配需前往WBO後台。
