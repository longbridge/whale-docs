# 系統介紹

# 概述

公告管理主要方便劵商運營人員對APP等交易終端,發出劵商面向客戶的系統相關公告。

# 公告管理說明

本作業主要用於維護面向客戶終端的系統公告管理

<img src="/assets/X9OfbC3ZoodHSUxYolfjbaGZpGb.png" src-width="3212" src-height="1612" align="center"/>

當需要新增一個對外公告時,可以點選右上角【新增公告】,開始配置一個新公告

<img src="/assets/ZUZxbRuE0obR2Px7gHMjYYkGp9k.png" src-width="2134" src-height="1518" align="center"/>

公告類型支持如下：

<img src="/assets/LsxEbyZYzo4F4dxM4wcjnnPFpmf.png" src-width="943" src-height="492" align="center"/>

支持市場：

```text
- 公告類型為：通用公告、常規休市、臨時休市、通道問題，支持設置；
```

<img src="/assets/E1XPb0HL9omwlvxAs5ajsoojpBb.png" src-width="1118" src-height="676" align="center"/>

標的範圍

```text
- 公告類型為：單股票，支持設置；
```

頁面範圍：

<img src="/assets/ZZDIbnmVVo9kF5xqK36jbeFPpoh.png" src-width="933" src-height="553" align="center"/>

其他根據需要配置公告內容：

<img src="/assets/O61dbs8JXoHl2XxmXEyjGCg3pvh.png" src-width="963" src-height="1380" align="center"/>

當新增一個系統公告管理後：

1. 新創建公告，提交工單審批，審批通過後才會生效；
2. 可以在記錄右側操作區進行【編輯】或【禁用】
