# 業務介紹

# 概述

出币（又稱「提币」「放币」）是指用戶將存放在交易平台或個人錢包中的加密貨幣，轉出至外部目標地址（如其他交易平台賬戶、個人數字錢包、他人收款地址等）的資產轉移行為。其核心特點與場景如下：

- <b>業務本質</b>：通過區塊鏈網絡廣播交易，實現數字資產所有權的跨地址轉移，需依賴平台審核與鏈上確認雙重機制保障安全。
- <b>常見場景</b>：資產劃轉（平台間轉移）、個人儲存（轉入硬件錢包）、交易支付（轉至對方地址）、法幣兌現（先轉至支持提現的平台）等。
- <b>關鍵規則</b>：需匹配幣種對應的區塊鏈網絡（如 USDT 的 ERC-20/TRC-20），且受平台 KYC 認證、提幣限額、安全驗證等規則約束。

# 出币步骤

出币業務的基本流程：

1. 前置準備：確認基礎條件
2. 發起出币申請（以交易平台為例）
3. 完成安全驗證與提交
4. WBO 工單審批流程
5. 等待平台審核與鏈上確認
6. 確認到账狀態
