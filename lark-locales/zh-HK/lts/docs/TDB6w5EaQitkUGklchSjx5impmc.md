# 2026-02-07 更新日誌

# 🎉 新功能

- 客戶端「我的費率」頁面支持自定義設置欄目
    - 路徑：業務參數設置-計費管理-客戶端展示
    - 注意事項
        - 數據更新後10分鐘內在客戶端生效
        - 新開通市場時，由交付人員開啟市場啟後(10分鐘內生效)，人工初始化。「普通訂單」為生效的第一條配置（詳細配置內容可參考其他市場），生效後不支持操作人員進行編輯
        - 免佣卡支持分市場處理，特殊收費欄目暫無法辨識。若某欄目不支持，可在欄目說明中進行補充
