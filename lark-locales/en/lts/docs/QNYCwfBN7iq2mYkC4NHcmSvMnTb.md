# Quick Start

You can quickly access the "Corporate Actions" → "Client Details" page via the sidebar. This page only supports viewing information for clients participating in corporate actions and allows inspection of exercise-related information and details to serve as a reference for traders regarding client exercises. To manage client corporate action allocations, please proceed to the WBO backend.
