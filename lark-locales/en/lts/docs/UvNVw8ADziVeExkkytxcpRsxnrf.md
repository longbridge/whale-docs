# Quick Start

# Overview

WTT provides users with the standard edition data for the Watchlist, Stock Detail, and Market pages, and offers a comprehensive component library that enables users to flexibly select and customize according to their actual needs, achieving an efficient and convenient operational experience.

# Market Data Types

WTT enables users to access multi-market market data. New users are granted delayed market data by default and may apply to upgrade to real-time market data. An overview of the market data levels supported by each market is as follows:
