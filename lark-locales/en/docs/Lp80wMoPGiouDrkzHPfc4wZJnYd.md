# Quick Start

# Overview

The Trade Inquiry module provides primary functions including fill order processing, IPO management, security restrictions, deal allocation, and order approval.

# Quick Start

## Functional Permission Requirements

You must ensure that your account has been granted the following permissions:

## Function Entry Points

The corresponding feature pages can be accessed through the WTT sidebar "Trade Inquiry" group or via the icon.

- When the sidebar is expanded: select the specific feature page directly through Trade Inquiry.
- When the sidebar is collapsed: access the desired feature page by selecting the corresponding icon.
