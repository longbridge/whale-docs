# Quick Start - Ordinary Bonus Shares and Dividends (BE)

# Applicable Scenarios

```text
Tenants directly connected to the Hong Kong Stock Exchange (with 05 file)

Mandatory-type BE (execution method: mandatory), including bonus shares, dividends, and equity distributions
```

# Preconditions

# Operating Instructions

## 1. Automatic Creation of Notices and Schemes

```text
Each morning, settlement staff import the previous afternoon's 05 file and 02 file.

After the files are successfully imported and parsed, the system will execute the automatic creation process; creation can be completed in approximately 10 minutes.

Based on the imported 05 file, the system will generate corporate actions with positions on the registration date.
```

## 2. Registration

```text
Operations on corporate actions can begin on the evening of the CCASS payment date (return of residual value) or on the following day.

    Example: If the CCASS payment date = 2024-02-05, execution may occur on the evening of the 5th or the morning of the 6th.

Click [Register]. Path: Corporate Actions - Corporate Actions - Details
```

## 3. Residual Adjustment

```text
If the funds paid by brokers need to be aligned with CCASS values, residual adjustments may be performed.
```

Click "Residual Adjustment"

## 4. Execution

```text
After successful registration (or after residual adjustments are completed), click Submit Execution to begin executing the corporate action.

Execution consists of two steps: execution submission and execution review; dual-person operation is recommended.
```
