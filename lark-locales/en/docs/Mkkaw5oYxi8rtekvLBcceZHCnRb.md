# Quick Start

# Overview

The Report Management System is a critical system for securities firms and financial institutions, used to process, manage, and generate various reports related to securities business. The system is designed to help firms ensure compliance with regulatory requirements and to provide accurate and timely financial and operational information.

# Quick Start

## Prerequisites

You must obtain the following authorizations before you can use the system functions normally.

### Step 1: Set up a print batch

- Click the [New] button in the upper right to add a batch print name and the composition of the batch reports.
- Add the reports composing the batch (select report name/print mode). Print criteria will automatically display relevant fields according to different reports and can be configured as needed.
- According to different report criteria fields, you can also set different conditions: on the batch details page, click [Add] to enter the report addition settings page, select a report, and set the report criteria.

Example: For date, the following options are available:

- ledger date â€” current accounting date
- Last ledger date â€” previous accounting date
- year start date â€” first day of the year
- month start date â€” first day of the current month
- last month end date â€” last day of the previous month
- last month begin date â€” last day of the previous month
- system date â€” current calendar date
- last system date â€” previous calendar date
- Before last ledger date â€” the ledger date before the previous one
- Before last system date â€” the calendar date before the previous one
