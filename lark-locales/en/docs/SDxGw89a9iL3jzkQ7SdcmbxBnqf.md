# 2024-02-27 Release Notes

# 🎉 New Features

- SMS Sending Records Query

    <img src="/assets/GU9mbqQffoIIzZxP1Ixjo3aTpQh.png" src-width="1868" src-height="1642" align="center"/>
    

    <img src="/assets/KxDObkw9Eohw52xsTdTjboaAp2f.png" src-width="3240" src-height="1624" align="center"/>
    - Supports viewing SMS sending records, including common information such as sending time, sending status, sending scenario, sending status description, and message content.
    - Path: "Customer Management System" - "Message Center" - "Message Records"
    - Permission: message.log_query — Message Sending Records Query (sensitive permission). This permission must be granted separately before the query operation can be executed.
