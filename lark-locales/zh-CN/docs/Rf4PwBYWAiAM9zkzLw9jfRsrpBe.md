# 🪀 改进与修复

- 出金规则 - 自动审核规则新增工单审批
    - 功能新增：自动审核规则的所有操作（含【新建】、【编辑】、【删除】及批量【启用 / 停用】），提交后均会新增工单审核。工单标题为「自动审核规则编辑」（工单标识：atm.withdraw_audits_edit）；上线后工单预设自动审批，若需人工审批，可在「审批流程配置」中修改。
    - 路径：业务参数设置 - 出金规则 - 自动审核规则
