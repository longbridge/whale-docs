# 快速上手

# 概述

所有已完成加白的客户钱包地址，均会在后台系统统一维护（包含地址资讯、认证时间、状态等），支援即时查询与管理，确保出入币操作的合规性与安全性。

<div class="callout callout-bg-5 callout-border-5 callout-color-1">
<div class='callout-emoji'>🎁</div>
<p>📍 菜单入口：WBO - 虚拟资产管理 - 出入币 - 渠道存管账户</p>
</div>

- 客户发起钱包地址认证前，系统会自动依据人工导入的「渠道存管账户」，为客户分配对应的账户。分配结果将做为后续认证、出入币操作的基础关联资讯，确保资金流向可追溯。

<img src="OI92bsI0YoAKMKx4eBFj51rYp3g" src-width="3778" src-height="1358" align="center"/>

<div class="callout callout-bg-5 callout-border-5 callout-color-1">
<div class='callout-emoji'>🎁</div>
<p>📍 菜单入口：WBO - 虚拟资产管理 - 出入币 - 客户钱包地址</p>
</div>

- 客户提交认证通过的钱包地址后，系统会将其纳入「加白名单」。所有已加白的地址将在后台统一维护，包含地址资讯、分配的存管账户、认证时间、状态等，支援即时查询与管理，保障操作合规与资金安全。
