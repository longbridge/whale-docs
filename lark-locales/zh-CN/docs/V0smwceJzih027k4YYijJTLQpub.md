# 快速上手

# 概述

WTT 组件工具箱中，交易相关组件使用操作入口

# 快速上手

- 通过 WTT 右上角组件工具箱 - 交易分类中找到交易功能相关组件，可将组件拖动至面板或独立使用
    <img src="NpWxbf6hPoKBAPxTRaJj9S3wpQg" src-width="600" src-height="128" align="center"/>
    <img src="P7BlbwULooGOMNxcyV1jLM10pJb" src-width="1354" src-height="904" align="center"/>

- 如无权限打开组件时，可提交权限申请，管理员在WBO后台审批通过后，再此刷新或重新登录即可使用组件功能
