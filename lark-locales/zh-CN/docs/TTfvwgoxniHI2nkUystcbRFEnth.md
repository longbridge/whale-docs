# 3. 证券转出／转入问题

# 3A 问：手工出仓如何修改转仓手续费？

**答：**在「证券管理＞证券取出＞取出明细」页面点击「修改转出费用」。

<div class="callout callout-bg-5 callout-border-5 callout-color-1">
<div class='callout-emoji'>🎁</div>
<p>提交手工出仓后，申请状态未点击提交审核处于 “<b>初始化状态”</b> 在「**取出明细」**能作修改，如已提交审核则不能修改。</p>
</div>

<img src="/assets/TXBfbRqzDoYDDTxZPvujzJgopFg.png" src-width="2496" src-height="574" align="center"/>

# 3B 问：如何处理实股转出及转入的申请？

**答：**在手工出仓／手工入仓，选择类型为实物股票点选为「是」（见 3B 图一、图二）。因实股需要作 OTCR 的申报，可在 `SDR070 - OTCR Report` 查看。

<div class="callout callout-bg-5 callout-border-5 callout-color-1">
<div class='callout-emoji'>🎁</div>
<p>处理实服申报／转出入股票前，请确认客户已同意 HKIDR（见 3B 图三）</p>
</div>

# 3C 问：如何修改「通知券商」的对手方或我方券商资料？

**答：**证券存入／取出中「通知券商」的资料（见 3C 图一、图二），在「证券管理＞券商信息配置＞我方券商信息／对手方券商信息」（见 3C 图三、图四）点击「编辑」可修改我方或对手方券商资讯。
