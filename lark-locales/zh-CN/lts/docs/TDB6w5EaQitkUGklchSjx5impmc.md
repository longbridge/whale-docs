# 2026-02-07 更新日志

# 🎉 新功能

- 客户端「我的费率」页面支持自定义设置栏目
    - 路径：业务参数设置-计费管理-客户端展示
    - 注意事项
        - 数据更新后10分钟内在客户端生效
        - 新开通市场时，由交付人员开启市场启后(10分钟内生效)，人工初始化。「普通订单」为生效的第一条配置（详细配置内容可参考其他市场），生效后不支持操作人员进行编辑
        - 免佣卡支持分市场处理，特殊收费栏目暂无法辨识。若某栏目不支持，可在栏目说明中进行补充
