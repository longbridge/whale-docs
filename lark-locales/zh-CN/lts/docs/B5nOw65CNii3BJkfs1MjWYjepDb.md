# 系统介绍

# 一、系统介绍

本指南针对加密货币入币业务的后台操作全流程，涵盖记录生成、查询跟踪、异常处理、撤销操作及退币流程，确保后台人员规范操作，保障业务合规与资金安全。

# 前置条件

- 后台操作人员使用专属账号密码登录 WBO 后台系统，进入「虚拟资产管理 - 出入币 - 入币」模块。
- 确认账号已具备「入币申请」「入币记录」「撤销」「退币」等操作权限，无权限需联系管理员申请。

## <b>入币操作</b>

<b>请求接收与记录生成</b>

<div class="callout callout-bg-5 callout-border-5 callout-color-1">
<div class='callout-emoji'>🎁</div>
<p>📍 菜单入口：WBO - 虚拟资产管理 - 出入币 - 入币申请</p>
</div>

- 客户端提交入币请求后，系统自动接收请求信息，并生成唯一入币记录（含记录编号、用户证券账号、入币币种、金额、目标地址、交易哈希 TXID、请求时间、状态等核心信息）。
- 若遇异常或有问题的入币申请，后台操作人员可直接在系统中执行删除操作。

<img src="LanObTVnboVFS5xfj7fj9DHypwh" src-width="3262" src-height="826" align="center"/>

<b>入币记录查询与跟踪</b>

<div class="callout callout-bg-5 callout-border-5 callout-color-1">
<div class='callout-emoji'>🎁</div>
<p>📍 菜单入口：WBO - 虚拟资产管理 - 出入币 - 入币记录</p>
</div>

- <b>多条件查询</b>：支持按记录编号、客户编号、创建时间 、状态、等条件筛选查询，快速定位目标记录。
- <b>详情跟踪</b>：点击单条记录查看详情，包含地址核对、交易哈希对应的区块确认进度等信息，实时跟踪入币全流程。

<img src="DuHRbCq7Oo0o1bx03o0jKdVEpCf" src-width="3268" src-height="806" align="center"/>

<b>未入币记录撤销操作</b>

- <b>操作前提</b>：仅针对状态为「钱包地址认证中」的记录（即区块链网络未开始确认、资产未转出的入币请求），其他状态记录不可撤销。
- <b>操作步骤</b>：
    1. 在入币记录列表中，找到目标撤销记录，点击「撤销」按钮；
    2. 系统弹出确认窗口，输入操作备注（如「用户重复申请」「地址填写错误」），再次确认撤销；
    3. 撤销成功后，记录状态更新为「已撤销」，系统自动留存撤销操作日志（含操作人、操作时间、备注），支持后续溯源。

<img src="XG2Db5drgo1wNGxr9aAjDC68p1c" src-width="3302" src-height="720" align="center"/>

<b>入币成功后退币流程</b>

- <b>操作前提</b>：仅针对状态为「已到账」的记录（即资产已成功入账至平台，用户申请退币或系统需强制退币场景），需核实用户身份及退币合理性。
- <b>操作步骤</b>：
    1. 进入「入币」模块，查询目标已到账记录，点击「申请退币」按钮；
    2. 填写退币信息：输入用户指定的退币地址（需与用户身份验证信息匹配）、选择对应网络（与入币网络一致）、核对退币金额（默认为入币金额，可调整，需注明调整原因）；
    3. 执行退币操作：后台点击「执行退币」，系统生成退币交易哈希，同步至区块链网络；
    4. 跟踪退币状态：实时查看退币记录的区块确认进度，确认完成后，更新入币记录状态为「已退币」，并留存退币全流程日志；
    5. 通知用户：退币成功后，需线下通知客户。
