# 🎉 新功能

- 短信发送记录查询功能

    <img src="/assets/WaN8bIxDYoVPFGxsRJwjlcFnptb.png" src-width="3240" src-height="1624" align="center"/>

    - 路径：「客户管理系统」-「消息中心」-「消息记录」
