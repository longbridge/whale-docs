# 12.1 批量报表打印

# 12.1A 问：批量打印财务日期的 Ledger date 跟 Last ledger date 差别是什么？

<img src="TxvxbdjiIoXe6axHieAjQZQqpQe" src-width="2388" src-height="956" align="center"/>

**答：**Ledger date 为系统财务日、Last ledger date 为上一天的账务日。

其他的账务日期选项：

- ledger date 当前账务日
- last ledger date 上一个账务日
- year start date 年初第一天
- month start date 本月初第一天
- last month end date 上月最后一天
- last month begin date 上月最后一天
- system date 当前自然日
- last system date 上一自然日

<div class="callout callout-bg-5 callout-border-5 callout-color-1">
<div class='callout-emoji'>🎁</div>
<p>见以下例子：ledger date（当前账务日）为 10 月 27 日；system date（当前自然日）为 10 月 31 日。</p>
</div>

# 12.1B 问：如何设置批量打印？

**答：**请参考以下步骤

(i). 于「报表管理＞批量打印」页面点击「新建」

<img src="LAWCbYQ5qo31LMxAniPjR859pRh" src-width="2640" src-height="990" align="center"/>

(ii). 自定义批次名称后，点击「确定」

<img src="KTVUbNrWPo3EAxxDW3djtGMRpee" src-width="2369" src-height="938" align="center"/>

(iii). 完成创建后，点击「详情」添加资料

<img src="C13UbVrh6oMKwbxhtCmj6fZ0pkb" src-width="2623" src-height="986" align="center"/>

(iv). 于右上方点击「添加」，可选择报表、文件类型及打印条件。把需要导出的报表逐份添加到批次。

<img src="LNJkb9hNPoyyWdxgP5wjWettpSc" src-width="2380" src-height="485" align="center"/>

<img src="LP1pbqomRogskRxRz4zj1ptbpXh" src-width="2194" src-height="1387" align="center"/>

<em>SDR 持仓类型的报表可指定单一客户导出资料，文件类型可选 CSV/PDF/XLS</em>
