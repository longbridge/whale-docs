# 系统介绍

# 一、系统介绍

本指南针对加密货币出币业务的后台操作全流程，涵盖记录生成、查询跟踪、工单审批流程，确保后台人员规范操作，保障业务合规与资金安全。

# 前置条件

- 后台操作人员使用专属账号密码登录 WBO 后台系统，进入「虚拟资产管理 - 出入币 - 出币」模块。
- 确认账号已具备「出币申请」「审批」等操作权限，无权限需联系管理员申请。

## 出币<b>操作</b>

<b>请求接收与记录生成</b>

<div class="callout callout-bg-5 callout-border-5 callout-color-1">
<div class='callout-emoji'>🎁</div>

- 客户端提交出币请求后，系统自动接收请求信息，并生成唯一入币记录（含记录编号、用户证券账号、入币币种、金额、目标地址、交易哈希 TXID、请求时间、状态等核心信息）。

- 客户提交的出币申请已自动对接 WBO 工单（工单标识：atm.va.coin_withdraw_apply），系统将依据预设的审批配置流开展审批。
