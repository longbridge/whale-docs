# 2024-11-11 更新日志

# 🎉 新功能

- 基金基本信息支持批量编辑
    <img src="SQ3ZbHwEkox2XBxPVE9jkNM8pFh" src-width="1280" src-height="832" align="center"/>

    - 路径：「基金管理」-「公募基金管理」

- 基金新增风险披露文件配置以及基金文件管理模块
    <img src="NA1Qbf4YYoQ7aZxo5lWjYnOHpRe" src-width="1280" src-height="830" align="center"/>

    - 路径：「基金管理」-「公募基金管理」-「编辑」/「详情」
    - 2.新增基金文件管理模块，支持上传基金相关的文件
    - 路径：「基金管理」-「公募基金管理」-「基金文件」
