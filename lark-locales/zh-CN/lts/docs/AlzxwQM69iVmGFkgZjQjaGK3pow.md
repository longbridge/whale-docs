# 2025-Q3 What‘s New

# v 2.33.0

# 🎉 新功能

- Order Book 中添加拒绝来源信息，可以更准确清楚的定位知晓订单是被系统拒绝还是上手券商拒绝。
    <img src="MJ43bkcbYoYSNlxAFXejcR4Xplh" src-width="2880" src-height="530" align="center"/>

- WTT 相关的操作日志均可在 WBO 中查看管理，WTT 上用户的操作行为（例如：导出、新增、编辑修改等）均可追踪到。
    <img src="DfPEbTVJIoFbMux59JNjGUcapyc" src-width="3820" src-height="1822" align="center"/>

- 交易组件内的资产数据优化，最大购买力修改为“最大可开仓金额”，该数据会通过最大购买力及当前标的的初始保证金比例来结合计算。
